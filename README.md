# RapidFire-AI-Winter-LLM-Challenge

## RAG challenge 
#### Dataset
The Docugami KG-RAG (Knowledge Graph Retrieval-Augmented Generation) dataset focusing on SEC 10-Q filings is a labeled, open-source dataset designed for testing RAG performance on complex business documents. [1, 2,]  
Key Details for Citation and Usage: 

• Dataset Name: Docugami KG-RAG - Sec 10-Q 
• Source: LlamaHub (LlamaIndex) 
• Content: Contains annotated SEC 10-Q documents for major tech companies, including questions and reference answers. 
• Access: Can be downloaded via LlamaIndex CLI:  
• Citation/Reference: Information is available via the Docugami blog (December 13, 2023) and LlamaHub. [2, 4]  

Performance Highlights: 

• Accuracy: In tests, the Docugami KG-RAG approach achieved 48% accuracy on the SEC 10-Q dataset, compared to 33% for standard OpenAI Assistants. 
• Methodology: Uses hierarchical semantic chunking to improve retrieval accuracy on long-form, complex business reports. [1, 3, 4]
